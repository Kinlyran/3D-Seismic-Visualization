# 3D-Seismic-Visualization #
Small GUI program to Plot a 3D seismic volume as slices along the x,y,z axis.Plots any regular 3D volume,requires IL,XL range and spacing.

## Requires: ##
* numpy
* tkinter
* obspy
* mayavi
  
## GUI :

* Enter xline and inline range
